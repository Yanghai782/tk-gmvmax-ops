# Troubleshooting Common Issues

## High CPM

### Is it actually a problem?
High CPM alone is not bad. Only problematic if CPM > CPM_cap (calculated from formula).

### Causes
1. Creative mismatch — CTR too low, system penalizes
2. Audience too narrow — bidding war, costs spike
3. System still learning — CPM naturally high in exploration phase

### Fixes (in order)
1. Replace creative (most effective)
2. Broaden audience targeting
3. If product GPM is inherently too low: raise price or drop product

## Flash Spend (Budget Burns Instantly)

### What it means
System is aggressively exploring audience. This is **normal during learning**.

### Actions
1. Do NOT cut budget (kills learning)
2. Observe 2-3 days, check if performance follows
3. If 3 days of flash spend + 0 conversions: kill the creative
4. If flash spend + conversions come in: system found signal, let it run

### Prevention
- Avoid peak hours for new campaigns (overnight often has less competition)
- But do NOT micro-manage time schedule — consistency matters more

## Zero Conversions

### Decision tree
- Day 1-2: Normal, especially if spend is low
- Day 3, 0 conversions, any spend level: kill creative
- Day 3, 0 conversions, high spend: kill immediately, wasted budget
- Day 3, 1-2 conversions, high spend: monitor, may be poor product-creative fit

## When To Create a New Campaign

### Create new campaign when:
- Creative style is completely different from existing
- Targeting a completely different audience
- Old campaign has run its course (creative decay, audience exhausted)

### Do NOT create new campaign when:
- Just tweaking an existing creative (add to same campaign)
- Same product, similar creative angle (add to same campaign)
- Wanting to test a slight audience variation (add to same campaign)

## Campaign Limit
GMV Max can run multiple campaigns simultaneously. System auto-distributes budget.
No hard limit, but more campaigns = thinner budget per campaign = slower learning.

## ROI Target Setting
- No fixed correct value — adjust based on data
- Cold start phase: set lower (or none) to prioritize learning
- Stable phase: adjust based on actual CPA and margin requirements
- If ROAS target is too aggressive: system may not spend at all

## Overnight / Off-Peak Hours
- If campaign is spending smoothly: let it run
- Frequent pausing = system learning reset
- Only pause if consistently wasting budget with no results

## Old Campaign Creative Refresh
- Old creatives from one campaign won't automatically match new creatives in same campaign
- If creative direction is significantly different: create new campaign
- If similar angle: add to existing campaign

## 800 Impressions, No Order — Panic?
- Check if creative is old/decayed (2-4 weeks)
- Check if ROAS target is too aggressive
- Consider creative refresh (new iteration, not completely different)
- 800 impressions with no conversion at high CTR = audience-product mismatch

## Core Rule
**Reduce frequency of changes. Give the system time to learn.**
