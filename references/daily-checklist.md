# Daily Operations Checklist

Run through these every day. Takes 5 minutes.

## Daily Must-Check

[ ] Is the campaign spending? No spend = something is broken
[ ] Any creatives with 3+ days and 0 conversions? Kill them
[ ] Are winning creatives still spending? Monitor for decay
[ ] Any new creatives added today? Keep pipeline flowing (1-2/day)
[ ] CPM above CPM_cap on any creative? Flag for elimination
[ ] Any pausing/restarting happened? Learning phase resets every time

## Weekly Review

[ ] Which creatives are performing? Rank by ROI, CPA, volume
[ ] Which creatives need to be retired? Near 2-4 week lifespan
[ ] Are winning creatives being cloned/iterated? Scale what works
[ ] Is CPA stable? If yes, can consider modest bid reduction
[ ] Is the audience pool exhausted? May need audience expansion

## Critical Warnings

**Learning Phase Protection:**
- During learning (first 50 conversions): DO NOT pause, restart, change bid,
  narrow audience, or otherwise touch the campaign
- Every interruption = system restarts learning from scratch
- This is why many advertisers never exit learning phase

**CPM Check:**
- If CPM > CPM_cap: creative is losing money, kill it
- High CPM alone is not bad — only bad if above your calculated cap
- CPM naturally fluctuates; look at trends, not single data points

**Creative Pipeline:**
- Always have 1-2 new creatives in testing
- Winning creative can be iterated: similar angles, same product
- One winning creative can carry multiple products in same category

**Flash Spend:**
- If budget burns instantly (minutes, not hours): system is exploring
- Normal during learning phase; observe 2-3 days
- If still flash-spending after 3 days with 0 conversions: kill the creative
- Do not cut budget during flash spend (kills learning)

## Quick Decision Matrix

| Situation | Action |
|-----------|--------|
| 3 days, 0 conversions | Kill creative |
| CPM > CPM_cap | Kill creative (or check if CPM trending down) |
| CPA > target * 1.5 | Monitor 2 more days; kill if no improvement |
| Not spending at all | Check bid, creative approval, product eligibility |
| Flash spending + conversions | Normal learning; observe |
| Flash spending + 0 conversions, 3 days | Kill creative |
| Campaign in learning, ROI volatile | Do nothing — this is normal |
